# 毛主席智慧宝库 Mao-wisdom-skill

> 毛泽东选集经典语录检索，以毛主席的思想和语气分析指导当代问题

## 项目简介

这是一个 OpenClaw Agent Skill，模拟毛主席的语言风格和思想方法，回答人生困惑、事业选择、战略决策、团队管理等各类问题。

## 核心功能

- **语录检索** — 从毛泽东选集经典语录库中按主题检索原文，注明出处
- **思想分析** — 以毛主席的辩证思维和战略高度分析当代问题
- **文风模拟** — 模拟毛主席特有的语气、词汇、句式风格进行回答

## 主题分类

| 主题 | 说明 |
|------|------|
| 战略全局观 | 星星之火可以燎原、藐视敌人、前途光明等 |
| 矛盾论与实践论 | 矛盾分析、实践检验、调查解决 |
| 群众路线 | 从群众中来到群众中去、人民创造历史 |
| 独立自主、自力更生 | 自己动手、立足自身力量 |
| 持久战思想 | 主动性、灵活性、计划性 |
| 斗争精神 | 奋斗、革命意志、克服困难 |
| 实事求是 | 具体问题具体分析 |
| 工作方法 | 弹钢琴、集中兵力、批评与自我批评 |
| 革命乐观主义 | 诗词豪情、革命必胜信念 |

## 文件结构

```
Chairman-Mao-wisdom-skill/
├── SKILL.md              # Skill 主文件（文风指南+回答结构）
├── REFERENCES.md          # 参考文档
├── mao-quotes.md         # 经典语录库（按主题分类）
├── iteration.sh          # 自动迭代脚本（每小时）
└── README.md             # 本文件
```

## 使用方式

在 OpenClaw Agent 中，当用户询问人生困惑、事业选择、战略决策、团队管理、困难应对等问题时，Agent 自动调用本 Skill 进行回答。

## 持续完善

本项目通过 `iteration.sh` 脚本每小时自动迭代，持续扩充：
- 更多经典语录和思想内容
- 更丰富的对话示例
- 更精准的文风模拟
