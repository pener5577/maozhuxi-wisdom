# 抖音AI生成・防崩高质量提示词模板

## 一、三段式万能公式（生成时必加）

### 1. 画质防崩（必加）
```
4K, ultra-detailed, cinematic lighting, masterpiece, best quality, sharp focus, realistic textures, vibrant colors, cinematic composition, dynamic camera, lens flare, particle effects
```

### 2. 动作防崩（必加）
```
smooth animation, fluid motion, dynamic pose, powerful impact, explosive visual effects, fast cutting, perfect rhythm, no distortion, no blurry, no messy lines
```

### 3. 抖音适配（必加）
```
vertical 9:16, high contrast, eye-catching, explosive special effects, blockbuster style, suitable for Douyin hot video
```

---

## 二、主体公式（每次只用这个结构）

```
主体 + 动作/变化 + 特效 + 光影
```

❌ 不要堆一堆无关内容
❌ 不要写太复杂

---

## 三、禁用词（一加就崩）

- 模糊、简单、低清、卡通、Q版、可爱、简约
- 太多颜色混写（红橙黄绿青蓝紫全写）
- 太抽象的词（"很帅""很牛""无敌"）

---

## 四、加这两个词直接更炸

- `explosive effects`（爆炸特效）
- `cinematic slow motion`（电影慢动作卡点）
