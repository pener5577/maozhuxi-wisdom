## 核心升级：故事分镜系统

**旧模式：** 批量生成相似风格图片，拼成幻灯片
**新模式：** 按故事情节分镜，每张图有明确的叙事任务

### 分镜规划表（每期视频必做）

在生成之前，先填写这个表格：

| 镜号 | 时间范围 | 叙事内容 | 画面主体 | 场景 | 情绪/氛围 |
|------|---------|---------|---------|------|---------|
| 1 | 0-3秒 | 故事开场 | 主角登场 | 江南书院 | 清新恬静 |
| 2 | 3-6秒 | 相遇相识 | 书院课读 | 书架前 | 好奇心动 |
| 3 | 6-9秒 | 情感萌发 | 树下共读 | 古树下 | 甜蜜温馨 |
| ... | ... | ... | ... | ... | ... |

**原则：40张图 = 40个不同叙事时刻，不要重复相似画面**

## 一、文生图（image-01）核心结构 v2

### 故事分镜提示词公式

```
【叙事内容】+【主体描写】+【场景细节】+【光线氛围】+【构图】
```

### 实战模板（梁祝化蝶为例）

```
【镜号3·共读】一位清秀书生与一位婉约女子共坐于古梅树下，
手执书卷，阳光透过枝叶在衣袂上洒落斑驳光影。
书生身着青色长衫，眉目清朗；女子着淡粉色襦裙，发间簪着小小梅花。
【场景】古梅虬枝、落花纷飞、远处青山如黛、近处青石书案
【光线】午后的温暖散射光，金色光斑，柔和阴影
【氛围】情窦初开的甜蜜与清新
【构图】竖屏9:16，前景古梅树干虚化，中景二人，远景青山
【质感】3D渲染，细腻丝绸纹理，柔和电影光，浅景深，低饱和高级灰调
```

### 叙事画面 vs 展示画面

| 类型 | 目的 | 提示词重点 |
|------|------|-----------|
| 叙事画面 | 推动情节 | 人物动作、表情、互动、环境变化 |
| 展示画面 | 展示美 | 人物特写、华丽场景、静态唯美 |

**人物动作（叙事专用）：**

## 二、TTS 语音优化 v2

### 平和叙事声音选择

**首选（故事讲述者）：**

| voice_id | 名称 | 特点 | 适合 |
|----------|------|------|------|
| `audiobook_female_1` | 有声书女1 | 温柔、讲述感、节奏平稳 | 女性视角故事、爱情传说 |
| `audiobook_male_1` | 有声书男1 | 温润、沉稳、叙事感强 | 历史典故、神话传说 |
| `Chinese (Mandarin)_Gentleman` | 温润男声 | 优雅内敛、不急不缓 | 古风叙事、诗词故事 |
| `Chinese (Mandarin)_Gentle_Senior` | 温柔学姐 | 柔和细腻 | 婉约风格 |

**避免使用：**
- `female-yujie`（太御姐，攻击性）
- `male-qn-badao`（太霸道）
- `female-tianmei`（太甜美，像儿童内容）

### 语速调整（关键！）

**问题：** 默认语速偏快，像在念稿
**解决：** 生成后用 FFmpeg 减速

```bash
# 0.85倍速：讲述感最强，适合古风故事
ffmpeg -i narration.mp3 -filter:a "atempo=0.85" narration_slow.mp3

# 0.9倍速：略微放慢，保持清晰
ffmpeg -i narration.mp3 -filter:a "atempo=0.9" narration_slow.mp3
```

### 旁白脚本写作规范

**旧模式：** 描述画面内容
**新模式：** 讲述故事，有起承转合

```
❌ 旧："一位身着淡粉色襦裙的女子，梳着精致的发髻，站在古梅树下。"
✅ 新："春日午后，书院中的古梅树下，她翻开书卷，他却偷偷望向她。"
```

**原则：**
- 每句旁白推动一个新的叙事信息
- 不要重复画面中已经看到的内容
- 用短句，有节奏感，适合讲述

## 三、文生视频（Hailuo-2.3）核心结构 v2

### 故事视频提示词公式

```
【叙事主体】+【场景环境】+【核心动作】+[运镜指令]+【光线氛围】
```

### i2v 模式特别注意

**只描述画面「动」的部分，不要描述静态内容！**

```
❌ 错误（描述静态）："古风书生的侧脸特写，端坐于书案前，窗外竹影摇曳"
✅ 正确（描述运动）："书生放下书卷，轻轻抬起头，目光望向窗外，竹叶随风轻摇，
光影在他脸上缓缓移动，[左摇]镜头缓缓向左平移，柔和的午后光线"
```

### 中国风叙事动作词库

```
轻抬眼眸、衣袂飘动、发丝微扬、书页翻动
落花飘零、雨丝斜落、水波荡漾、云雾流动
回眸一笑、执手相望、比翼双飞、化蝶纷飞
```

## 四、质量检查清单

### 生成前
- [ ] 完成分镜规划表（40个叙事时刻不重复）
- [ ] 每张图有明确的叙事内容和情绪基调
- [ ] 旁白脚本与画面叙事一一对应

### 生成后
- [ ] 图片按叙事顺序检查：是否符合故事发展？
- [ ] 是否有两张图叙事重叠/相似？
- [ ] TTS 语速是否平和？（播放听一下）
- [ ] 旁白和画面是否对应？（旁白说"离别"时画面是送别场景）

### 上传前
- [ ] 视频整体叙事流畅，从头到尾有故事感
- [ ] 色调统一（低饱和高级灰）
- [ ] BGM 音量适当（不超过旁白 25%）

## 五、常见问题

### Q: 图片风格不一致怎么办？
**A:** 在提示词中加入固定的风格锚点：
```
【统一风格锚点】所有画面遵循：3D渲染、柔和电影光、低饱和灰调、东方意境美学
```

### Q: 旁白太快像念稿怎么办？
**A:** 脚本用短句 + FFmpeg 减速 0.85x

### Q: 故事叙事不连贯怎么办？
**A:** 先写完整的旁白脚本，再按脚本分镜生成图片，保证图和话对应
